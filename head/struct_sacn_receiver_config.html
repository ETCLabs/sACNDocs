int SacnReceiverConfig::source_count_max

The maximum number of sources this universe will listen to.

May be kSacnReceiverInfiniteSources. When configured to use static memory, this parameter is only used if it's less than SACN_RECEIVER_MAX_SOURCES_PER_UNIVERSE – otherwise SACN_RECEIVER_MAX_SOURCES_PER_UNIVERSE is used instead.
