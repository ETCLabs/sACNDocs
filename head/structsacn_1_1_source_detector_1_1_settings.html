◆ source_count_max

int sacn::SourceDetector::Settings::source_count_max {kSacnSourceDetectorInfinite}

The maximum number of sources the detector will record.

It is recommended that applications using dynamic memory use kSacnSourceDetectorInfinite for this value. This parameter is ignored when configured to use static memory – SACN_SOURCE_DETECTOR_MAX_SOURCES is used instead.

◆ universes_per_source_max

int sacn::SourceDetector::Settings::universes_per_source_max {kSacnSourceDetectorInfinite}

The maximum number of universes the detector will record for a source.

It is recommended that applications using dynamic memory use kSacnSourceDetectorInfinite for this value. This parameter is ignored when configured to use static memory – SACN_SOURCE_DETECTOR_MAX_UNIVERSES_PER_SOURCE is used instead.
