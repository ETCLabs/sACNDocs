◆ manually_process_source

bool SacnSourceConfig::manually_process_source

If false (default), this source will be added to a background thread that will send sACN updates at a maximum rate of every 23 ms.

If true, the source will not be added to the thread and the application must call sacn_source_process_manual() such that all needed start codes are sent at its maximum DMX rate, typically 23 ms.
